Términos y Condiciones de Uso

1. Identificación del proyecto

Nombre: Vivero Quilino
Versión: 1.0
Autores: Augusto Aguirre, Priscila Córdoba, Gabriel Córdoba, Martín Miranda, Rocio Medran
Docente: Karina Salto
Institución: Quality ISAD

El objetivo principal de este sistema es servir como un catálogo digital y punto de contacto moderno y eficiente para el vivero municipal de la ciudad de Quilino, Córdoba.

Se enfoca en maximizar la accesibilidad y la conversión de clientes al:

  • Brindar una interfaz clara e intuitiva para explorar el catálogo de productos y servicios.
  • Permitir visualizar detalles relevantes de cada artículo, como disponibilidad y características.
  • Integrar comunicación directa vía WhatsApp para consultas y compras.
