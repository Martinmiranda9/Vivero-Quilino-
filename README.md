# Vivero-Quilino-
El proyecto "Vivero Quilino" es una aplicación web para el vivero municipal. Resuelve la falta de un catálogo digital, optimiza la gestión interna (CRUD de productos, categorías, servicios) y mejora la comunicación con clientes (pedidos por WhatsApp). Su objetivo es optimizar la gestión y mejorar la experiencia de usuario.
